# Alwin & Annmareena Wedding Website — Version 2

Replace your current repository files with these files, then run:

```bash
git add .
git commit -m "Upgrade wedding website to version 2"
git push
```

GitHub Pages will redeploy automatically.

Version 2 adds refined loading animation, scroll progress, cinematic hero polish, enhanced music control, upgraded cards, venue labels, and mobile refinements.


## Version 3

- Fixed Annmareena cropping on mobile.
- Fixed mobile browser viewport fill.
- Rings image is now the hero.
- Church image remains in the venue section.
- Quote uses Alwin & Annmareena.
- Footer remains With love, Alwin and Ana.
- Added a native RSVP form with Google Sheets integration.
- See `RSVP_SETUP.md`.


## Version 4 changes

- Alwin and Annmareena now use the same visual font size on mobile.
- Long-name fitting uses subtle horizontal compression rather than shrinking one name.
- Added `viewport-fit=cover` and iPhone safe-area handling.
- Top and bottom browser areas blend into the espresso theme where supported.
- Venue cards use a consistent 16:10 image frame and equal visual height.
- Added a subtle espresso overlay to normalize the two venue photographs.
- Venue cards are shorter and more balanced.


## Version 5 changes

- Added a custom A&A monogram treatment.
- Added the personal line: `From Waterloo to forever.`
- Refined venue and RSVP buttons.
- Added personalized RSVP confirmation messages.

## Version 7 changes
