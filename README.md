# Alwin & Annmareena Wedding Website

A responsive one-page wedding website ready for GitHub Pages.

## Preview locally

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files and folders from this project.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`.
6. Save and wait for GitHub to publish the site.

## Replace the hero image later

Replace:

`assets/images/rings-hero.jpg`

Keep the same filename, or update the image path in `styles.css` and `index.html`.

A wide landscape image works best. Aim for at least 1800 × 1200 px.

## Music notice

The included MP3 is the uploaded testing track. Replace it with music you have permission to publish before sharing the website publicly.

Replace:

`assets/audio/wedding-song.mp3`
